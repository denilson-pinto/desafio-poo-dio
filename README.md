# **Desafio de projeto: Abstraindo um Bootcamp Usando Orientação a Objetos em Java**.

Projeto que compõe parte do conteúdo do bootcamp **GFT Quality Assurance #1**, realizado na plataforma de ensino online [Digital Innovation One](https://www.dio.me/). 

Neste projeto foi realizado uma pequena simulação de um bootcamp como forma de aprendizado e prática dos conceitos pilares do paradigma de programação Orientação a Objetos, tais quais:

* Abstração;
* Encapsulamento;
* Herança;
* Polimorfismo.
